## Install

Download the Node.js SDK using your favorite package manager.

```bash
npm install @kobbleio/admin-sdk
```

## Quick Start

```typescript
import { Kobble } from '@kobbleio/admin-sdk'

const main = async () => {
    const kobble = new Kobble('YOUR_SECRET_KEY');
    const whoami = await kobble.whoami();

    console.log(whoami);
}
```
