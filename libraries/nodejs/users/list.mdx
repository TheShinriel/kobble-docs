---
title: 'List'
---

You can retrieve all users using the `list` method.

## Example usage

```typescript
import { Kobble } from '@kobbleio/admin'

const main = async () => {
    const kobble = new Kobble('YOUR_SECRET_KEY');
    const users = await kobble.users.list({
        limit: 10,
        page: 1,
    });

    console.log(users);
}
```
