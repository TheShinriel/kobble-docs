## Install

Download the Python SDK using your favorite package manager.

```bash
pip install kobble_admin
```

## Get your secret key

You can obtain your secret key from [your dashboard](https://app.kobble.io/p/sdk/secrets).

![Secret key page](/images/sdk-secret-page.png)

## Test your integration

```python
from kobble_admin import Kobble

kobble = Kobble("YOUR_SECRET_KEY")
whoami = kobble.whoami()

print(whoami)
```
